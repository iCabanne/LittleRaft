Menu Styles

Menus come in all shapes and sizes. Through the wonder of CSS we can take a simple HTML list and transform it into exactly what we want. Media queries give further flexibility to change the menu to suit the media. These examples draw from many examples and demos posted by generous developers. (I've just started on this page so there is a lot more to come.)

The Basic HTML

<ul>
   <li><a href="javascript:void(0);">One</a>
      <ul>

   </li>

4. Gradient bar with hover drop down

In the HTML the first line is changed to: <ul id="grad" class="blue">. The "grad" CSS controls the menu functionality and basic layout. The "blue" CSS controls color, borders and other design-only details. This menu has also been placed on a background that makes it pop.
